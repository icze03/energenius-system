# **App Name**: Energenius

## Core Features:

- Consumption Visualization: Display real-time energy consumption data in a clear, understandable format (e.g., graphs, charts).
- Target Setting: Allow users to set target consumption levels for different time periods.
- Reporting & Analysis: Provide summaries and comparisons of energy use vs. historical data and set targets.
- Smart Recommendations: AI-powered energy saving tips and recommendations tailored to the SME's usage patterns using an agent as a tool.
- Device Scheduling: Enable users to define schedules for energy-consuming devices (lights, HVAC) to optimize usage.

## Style Guidelines:

- Primary color: Deep teal (#468499) to represent efficiency and environmental consciousness.
- Background color: Light, desaturated teal (#E0F4F7) to create a calm and professional feel.
- Accent color: Warm orange (#FFA07A) to highlight key data points and calls to action, representing energy.
- Body and headline font: 'Inter' (sans-serif) for a modern, readable, and neutral look.
- Use clean, minimalist icons to represent energy-related concepts and device types.
- Design a clean and intuitive dashboard layout for easy access to information and key functions.
